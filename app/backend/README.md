## Getting Started

Follow these instructions to get the backend application up and running on your local machine for development and testing purposes.

## Prerequisites

- [Python](https://www.python.org/downloads/) (version specified in `pyproject.toml`)
- [pip](https://pip.pypa.io/en/stable/installation/)

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/your-repo.git
    cd your-repo/app/backend
    ```

2. Install the required dependencies:

    ```sh
    pip install -r ../../requirements-dev.txt
    ```

## Running the Application

To start the backend application, run:

python main.py
